useMemo란
"메모이제이션" 기법을 기반으로 불 필요한 연산을 최적화하는 리액트 훅

자매품 : useCallback 

memoization이란?
기억해두기, 메모해두기 라는 뜻 

반복적으로 수행해야하는 동일한 연산을 해야한다면 
매번 결과값을 다시 계산하지 않고 최초로 계산한 값을 메모리어딘가에 저장해두고 
필요 시 저장된 데이터 값을 사용하므로써 프로그램 성능이 나빠지는 것을 방지.

ex: 근사한 식당 - 메뉴가 독특함 - 한번씩뭔지 물어봄 - 메뉴판을 다시보고 대답해주는 것이 아니고 - 메뉴의 이름을 외워둠 - 머리속 기억을 이용해 대답 


React.memo
컴포넌트를 인수로 받아, 최적화된 컴포넌트로 만들어 반환. 
const MemoizedComponent = memo(Component);
// 반환값: 최적화된 컴포넌트, 
props를 기준으로 메모이제이션 된다. 
// 부모가 리렌더링이 되어도 자식이 받는 props가 바뀌지 않으면 다시 리렌더링 되지 않도록 메모이제이션 되므로 자동으로 최적화된다. 

// 인수: 컴포넌트.
